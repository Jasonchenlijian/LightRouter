# LightRouter
轻量级的路由框架，可用于组件间数据交互，页面跳转等

### Maven

	<dependency>
       <groupId>com.clj.router</groupId>
       <artifactId>lightrouter</artifactId>
       <version>1.0.0</version>
	   <type>pom</type>
	</dependency>

### Gradle

	compile 'com.clj.router:lightrouter:1.0.0'
